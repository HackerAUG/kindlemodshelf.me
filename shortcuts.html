Home Screen Shortcuts

Copy only the shortcuts you want into /mnt/us/documents. Tap them from the Kindle Home screen to run the corresponding app directly — no KUAL required.

Quick steps
  1. From this website, open the downloads/ folder and pick the script(s) you want.
  2. Copy them to /mnt/us/documents on your Kindle.
  3. Make them executable if necessary: 
    chmod +x /mnt/us/documents/*.sh
  4. Tap the file on the Home screen to launch the app.
Shortcut files
Alpine.sh Alpine mail client launcher — drop into /mnt/us/documents. KOReader.sh KOReader launcher — opens KOReader directly from the Home screen. KindleFetch.sh KindleFetch utility launcher. Kterm.sh KTerm terminal launcher.
Notes
  • All files referenced on this page live in downloads/ on the website — copy the specific ones you need.
  • If a shortcut doesn't run: confirm the file is executable and located in /mnt/us/documents.
  • These scripts are simple launchers that call the actual app start scripts on-device. Inspect them before use if you want to confirm exact behavior.
