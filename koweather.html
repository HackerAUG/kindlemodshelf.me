KOWeather — lo-fidelity Weather for KOReader

Did you say weather? Did you say e-reader? Did you say now? Say "welcome" to your new best friend, KOWeather, the lo-fidelity weather app that delivers weather conditions and forecasts to your favorite e-ink device. Weather information is provided by WeatherAPI.

Features

  • Forecast for current day
  • Forecast for the next 2 days
  • Hourly forecast for the current day
  • Celsius and Fahrenheit (except it's not yet working!)
  • 24- or 12-hour clock

Customization

Have it your way. This plugin lets you set the forecast location by postal code. Change the postal code in the plugin settings to get weather for your area.

Compatibility

  • Tested with the Kobo Libra H2O.
  • Other devices are unknown — feel encouraged to try this plugin on your device.
  • If it does or doesn't work on your device, our elves would love to know (open an issue on the GitHub repo).

Installation

Easy as pie — drag and drop:

  1. Download the latest release from the project's GitHub: roygbyte/weather.koplugin.
  2. Unzip the downloaded archive on your computer.
  3. Transfer the weather.koplugin folder to your KOReader-powered device and place it under /mnt/us/.adds/koreader/plugins/ (or /.adds/koreader/plugins/ depending on your device layout).
  4. Restart KOReader. The plugin should appear under Plugins → More tools → Weather (or a similar menu entry) — select it to run.

How to Use

  1. Open KOReader.
  2. Navigate to Plugins → More tools → Weather.
  3. View the current conditions and the short forecast. Tap or press the navigation control to return to KOReader.
  4. To change location, open the plugin settings and enter your postal code.

Notes & Troubleshooting

  • Weather data courtesy of WeatherAPI; an internet connection is required to fetch updates.
  • If the Celsius/Fahrenheit toggle isn't functioning yet, try updating the plugin or filing an issue on GitHub.
  • For device-specific quirks, open an issue on the repo with device model and KOReader version.

Contribute / Report Issues

Found a bug or want to improve KOWeather? File an issue or submit a pull request on GitHub: roygbyte/weather.koplugin.
