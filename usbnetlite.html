usbnetlite — Simple install & connection (Kindle)

Minimal, corrected install instructions so you can enable USB networking and control usbnetlite from KUAL.

Installation Instructions (only what you NEED)

  1. Download the appropriate update.bin (the MR package) for your firmware from the project's Releases page.
  2. Do not extract or unzip the file — you need the raw update.bin.
  3. Place the update.bin into a folder named mrpackages on your Kindle's USB storage (create mrpackages if it doesn't exist).
  4. On the Kindle, open KUAL and install the MR package via Helper → Install MR Packages (or run the MR installer from KUAL and pick the .bin in mrpackages).
  5. Optional/diagnostic: run ;log mrpi from the KUAL search bar to view installer logs if you need troubleshooting info.
  6. After installation, control / launch usbnetlite from the KUAL menu.

Connection Info (default config)

There is a config file you can edit with KOReader. The default connection credentials are:

username: root
password: kindle

Once usbnetlite is running you should be able to control it from KUAL.

Quick Launch

  • Install the MR package as above.
  • Open KUAL → find usbnetlite in the menu → launch.
  • Use the KUAL interface to enable/disable USB networking as needed.

Notes

  • This page contains only the minimal, corrected steps — follow the project's Releases for specific firmware builds.
  • If an installer log is required for debugging, use ;log mrpi in the KUAL search bar to read it.
